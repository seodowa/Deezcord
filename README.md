# CS323 Finals PIT: Deezcord

Deezcord is a real-time chat system where users can join specific chat rooms, explore multiple channels, and message with other people in real time. 

This project serves as our Performance Innovative Task in the course Parallel and Distributed Computing. This system aims to demonstrate concurrency through the use of WebSockets and a multi-channel architecture, which enables full-duplex, bidirectional communication over a single, persistent connection.
